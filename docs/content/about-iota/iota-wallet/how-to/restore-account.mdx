# Restore From Seed or Stronghold

If you have lost your mnemonic, you can still recover your account provided you have access to your seed or
`.stronghold` file.

<Seed />

## Back Up a Seed From a Firefly Profile

You can export you seed by clicking on the `...` menu, and then clicking `Backup Seed`.

   ![Backup Seed](/img/about-iota/iota-wallet/how-to/restore-account/backup-seed-falafel.png)

Firefly will then prompt you to enter your password.

   ![Enter Password](/img/about-iota/iota-wallet/how-to/restore-account/enter-password.png)

   ![Copy seed](/img/about-iota/iota-wallet/how-to/restore-account/copy-seed.png)

With your seed at hand, you can [import your wallet using the IOTA Web Wallet](../getting-started.mdx#importing-an-existing-wallet).

## Back Up a Seed From a `.stronghold` file

:::info Not Recommended

If you are using Firefly, we recommend using the [Backup Seed](#back-up-a-seed-from-a-firefly-profile) option from the Firefly menu.
Using a stronghold file directly should only be done if you don't have a profile in Firefly or if you are using a different wallet.

:::

You can use your `.stronghold` backup file to obtain your wallet's seed.

If don't know where your `.stronghold` file is, click on the `...` menu and select the `Copy system profile location`. 

   ![Copy Stronghold Location](/img/about-iota/iota-wallet/how-to/restore-account/copy-stronghold-location.png)

Next, you should click on  `Electron > Backup Seed`. 

   ![Backup Seed](/img/about-iota/iota-wallet/how-to/restore-account/backup-seed-menu.png)

Once you have the `.stronghold` file at hand, you can drag and drop it into Firefly's Stronghold Backup.

   ![Drag and Drop Stronghold](/img/about-iota/iota-wallet/how-to/restore-account/drag-and-drop-stronghold.png)

Firefly will then prompt you to enter your password to unlock the `.stronghold` file.

   ![Enter Password](/img/about-iota/iota-wallet/how-to/restore-account/enter-password.png)

Firefly will then show you the seed it restored from the `.stronghold` file. 

   ![Copy seed](/img/about-iota/iota-wallet/how-to/restore-account/copy-seed.png)

With your seed at hand, you can [import your wallet using the IOTA Web Wallet](../getting-started.mdx#importing-an-existing-wallet).
