# Manage NFTs and Digital Assets

You can access the Assets view by clicking on the `Assets` button on the left side menu.

   ![Click Assets](/img/about-iota/iota-wallet-dashboard/how-to/assets/click-assets.png)

You can use the buttons to filter by `Visual` and `Other` assets.

   ![Assets View](/img/about-iota/iota-wallet-dashboard/how-to/assets/assets-view.png)

## Send an NFT

To send an NFT:

1. Click on the desired NFT in the Assets view.
2. Click `Send`
